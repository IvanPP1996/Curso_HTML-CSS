**UTF-8:** Es un sistema de codificación de caracteres que traduce los caracteres (letras, símbolos, emojis, etc.) a bytes para que la computadora pueda almacenarlos, interpretarlos y mostrarlos correctamente.

## Los bytes

**¿Que es un bit?**

Es una unidad pequeña de información que tiene solo dos valores 1 o 0

**¿Qué es un byte?**

Es un conjunto de 8 bits se usan para representar información digital (letras, numeros, imagenes, etc)

    Ejemplo:

    A = 01000001

**Tamaños de almacenamiento**

    1 byte = 8 bits     (Un caracter, numero, etc)
    1 KB   = 1024 bytes (Página de texto simple)
    1 MB   = 1024 KB    (Una canción, una foto de buena calidad)
    1 GB   = 1024 MB    (Películas, video juegos pequeños)
    1 TB   = 1024 GB    (Disco duros, servidores)
